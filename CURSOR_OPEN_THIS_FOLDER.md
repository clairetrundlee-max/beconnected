# Open this folder in Cursor

The **Connected** app (including Friends → **Select time**) lives **here**:

**`/Users/clairetrundle/Downloads/connected`**

If Cursor still shows an old project (e.g. **inkind - AI Practice**) or **Calendar** on Friends instead of **Select time**:

1. **File → Open Folder…** (or **Open…**)
2. Go to **Downloads → connected** and choose that folder
3. In Terminal: `cd` to that same folder, then `npm run dev`
4. Hard-refresh the browser (**⌘⇧R**)

If you cloned **connected2** from GitHub, open **that clone folder** in Cursor instead, then copy the updated files from `Downloads/connected` or pull after you push.
